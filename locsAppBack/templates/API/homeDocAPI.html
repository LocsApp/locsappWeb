Locsapp API V1 Documentation :


Free-access endpoints

POST : /api/v1/rest-auth/registration/ - Subscribe a new user

  • email - EmailField
  • username - CharField
  • password1 - CharField
  • password2 - CharField

POST : /api/v1/rest-auth/login/ - Log a user in

  • username - CharField
  • password - CharField

Authorization Header required

POST : /api/v1/rest-auth/logout/ - Logout a User

GET : /api/v1/rest-auth/user/ - Request the user profile
