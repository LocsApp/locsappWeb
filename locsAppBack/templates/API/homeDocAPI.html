Locsapp API V1 Documentation :

POST : /api/v1/rest-auth/registration/ - Subscribe a new user

  • email - EmailField
  • first_name - CharField
  • last_name - CharField
  • username - CharField
  • password1 - CharField
  • password2 - CharField
  • birthdate - CharField
  • phone - CharField
  • living_address - CharField
  • billing_address - CharField
  • logo_url - CharField
  • is_active - BooleanField (default = false)
